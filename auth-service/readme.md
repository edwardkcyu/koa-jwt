# Auth Service

## Usage

### PRODUCTION START

```
yarn start
```

### DEVELOPMENT START

```
yarn dev
```

### UNIT TEST

```
yarn test
```

### BUILD DOCKER IMAGE

```
yarn docker:build
```

### PUSH DOCKER IMAGE

```
yarn docker:push
```

### Environment Configuration

You can use `.env` file, to configure project like this:

```
NODE_ENV=development
PORT=6010
JWT_SECRET=jioasjdfp
```
